## 📝 Описание

Современный адаптивный лендинг для продажи рыбы и морепродуктов. Сайт содержит каталог товаров, блок преимуществ, контактную информацию и призывы к действию.

## 🖼 Демонстрация

Сайт включает следующие блоки:
- **Шапка** с навигацией и кнопкой звонка
- **Главный экран** с заголовком и призывом к действию
- **Каталог** из 4 карточек товаров
- **Преимущества** (4 блока)
- **Контакты** с телефоном, почтой и режимом работы
- **Подвал**
